# Pricing Page Clarity Guidelines

**Date:** 2025-01-27  
**Purpose:** Ensure all pricing descriptions are user-friendly, honest, transparent, and clearly indicate features are included with no additional costs

---

## 🎯 Core Principles

1. **User-Friendly Language** - Conversational, not formal or legal-sounding
2. **No Overselling** - Be honest about what you actually provide
3. **No Unsubstantiated Claims** - Remove claims that can't be proven
4. **Transparency** - Clearly explain HOW features work, not just WHAT they are
5. **No Legal Language** - Avoid binding commitments and legal terminology
6. **Cost Clarity** - All features clearly included with no additional costs

---

## 🎯 Core Principle

**All features listed in a plan must be clearly stated as included in the subscription price with no additional costs.**

---

## ❌ Problematic Language (Avoid)

### Examples of Language That Could Imply Additional Costs:

1. **"Priority email support - Faster response for security issues"**
   - ❌ Problem: Could imply support is a paid add-on
   - ✅ Better: "Priority Email Support (Included) - Faster response times for security issues. Included in your subscription - no additional support fees."

2. **"Professional management - Service catalog managed by experts"**
   - ❌ Problem: Could imply professional services cost extra
   - ✅ Better: "Expert-Curated Service Catalog (Included) - Our service catalog and risk assessments are continuously updated and managed by cybersecurity and legal professionals. This professional oversight is included in your subscription at no extra cost."

3. **"Advanced analytics dashboard"**
   - ❌ Problem: Unclear if it's included or an add-on
   - ✅ Better: "Advanced Privacy Analytics Dashboard (Included) - Track privacy scores over time with full history. All included in subscription."

4. **"Unlimited exports"**
   - ❌ Problem: Could imply per-export fees
   - ✅ Better: "Unlimited Exports (Included) - Generate unlimited reports in PDF, Excel, CSV, and JSON formats. No per-export fees."

---

## ✅ Recommended Language Patterns

### Pattern 1: Explicit Inclusion Statement
```
"[Feature Name] (Included) - [Description]. Included in your subscription - no additional cost."
```

**Example:**
- "Priority Email Support (Included) - Faster response times for security issues. Included in your subscription - no additional support fees."

### Pattern 2: Clear No-Fee Statement
```
"[Feature Name] - [Description]. All included - no [type] fees."
```

**Example:**
- "Unlimited Exports - Generate unlimited reports in PDF, Excel, CSV, and JSON formats. No per-export fees."

### Pattern 3: Professional Service Clarification
```
"Expert-Curated [Service] (Included) - [Description]. This professional oversight is included in your subscription at no extra cost."
```

**Example:**
- "Expert-Curated Service Catalog (Included) - Our service catalog and risk assessments are continuously updated and managed by cybersecurity and legal professionals. This professional oversight is included in your subscription at no extra cost."

---

## 📋 Feature Description Checklist

For each feature description, ensure:

- [ ] Feature name is clear
- [ ] Description explains what it does
- [ ] Explicitly states it's included (if premium)
- [ ] No language implying additional costs
- [ ] No ambiguous terms that could suggest fees
- [ ] Clear statement: "included", "no additional cost", "no fees", etc.

---

## 🔍 Red Flag Words to Avoid

### Words That Imply Additional Costs:
- "Priority" → Just say "Email Support" or "Support"
- "Professional" → Avoid unless you can prove it
- "Advanced" → Use "Detailed" or "Full" instead
- "Unlimited" → OK, but clarify "no per-[unit] fees"
- "Optional" → OK, but clarify "no additional charge"
- "Premium" → Avoid marketing language

### Words That Oversell or Create Legal Risk:
- "Professional-grade" → Too subjective
- "Expert-curated" → Implies ongoing expert involvement
- "Managed by professionals" → Creates false expectations
- "Worth $X" → Unsubstantiated claim
- "Save X hours" → Can't guarantee
- "Guaranteed" → Legal commitment
- "Assured" → Legal commitment
- "Certified" → Requires certification
- "Professional oversight" → Legal-sounding, implies ongoing service

## ✅ Good Examples

### Support Features
```
✅ "Email Support - Contact us by email if you need help. We'll respond as quickly as we can, especially for security issues. Included in your subscription."
```

**Why:** Honest about what support actually is (email), no false promises about response times, no "priority" claims that can't be substantiated.

### Professional Services
```
✅ "Updated Service Catalog - Our catalog of 200+ services is regularly updated with the latest privacy information we can find. Included in your subscription."
```

**Why:** Honest about what it is (curated data that gets updated), no false claims about "expert management" or "professional oversight", no legal-sounding language.

### Unlimited Features
```
✅ "Unlimited Privacy Assessments (Included) - Run as many assessments as you need (all 3 types: Complete, Risk Exposure, Rights Checkup) with no monthly limits. All included - no per-assessment fees."
```

### Optional Features
```
✅ "Optional Encrypted Data Export (Included) - Export your privacy data in encrypted format (AES-256-GCM) for secure manual backup to your own storage. No additional cost."
```

### Advanced Features
```
✅ "Advanced Privacy Analytics Dashboard (Included) - Track privacy scores over time, view assessment history, monitor risk level changes, and export comprehensive analytics reports. Fully included in subscription."
```

## 🎯 Section-Level Clarifications

### At the Top of Each Plan Section

**Basic Plan:**
```
"All features listed below are included at no cost - completely free forever."
```

**Premium Plan:**
```
"All features listed below are included in your $2.99/month subscription. No additional fees, no hidden costs, no per-use charges."
```

### At the Bottom of Each Plan Section

**Basic Plan:**
```
"Everything listed is free forever. No signup fees, no hidden costs."
```

**Premium Plan:**
```
"No Hidden Fees: Everything listed is included in your $2.99/month subscription. No additional charges for support, professional services, or feature usage."
```

## 📝 Translation Considerations

When translating to Spanish and French:

- Ensure "Included" / "Incluido" / "Inclus" is prominently displayed
- Use clear "no additional cost" / "sin costo adicional" / "sans coût supplémentaire" language
- Maintain the same clarity about included features
- Avoid ambiguous terms in other languages

---

## 🔄 Review Process

Before publishing any pricing page updates:

1. ✅ Read each feature description
2. ✅ Check for any language that could imply additional costs
3. ✅ Verify "included" or "no additional cost" language is present
4. ✅ Test with someone unfamiliar with the product
5. ✅ Ask: "Could this be interpreted as costing extra?"
6. ✅ If yes, add clarification
