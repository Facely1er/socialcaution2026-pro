# Standalone HTML Demo

## Overview

`standalone-demo.html` is a **completely standalone HTML file** that can be opened directly in any browser without a build system, server, or dependencies.

## Features

✅ **Single File** - Everything in one HTML file
✅ **No Dependencies** - No React, no build system, pure vanilla JavaScript
✅ **Works Offline** - Can be opened from file system
✅ **Demo Ready** - Perfect for quick demos and sharing
✅ **Lightweight** - ~15KB file size

## Usage

### Option 1: Direct File Open
1. Open `standalone-demo.html` in any browser
2. That's it! No server needed

### Option 2: Share as File
1. Send the HTML file to anyone
2. They can open it directly
3. Works on any device with a browser

### Option 3: Host on Static Server
1. Upload to any static hosting (GitHub Pages, Netlify, etc.)
2. Access via URL
3. No build step required

## Limitations

This is a **simplified demo version**:
- Basic calendar functionality
- Simplified task structure
- No export functionality
- No score tracking
- No complex calendar generation

For full functionality, use the React version at `/privacy-calendar-standalone`
