# App Store download badge images

Official badge assets used on the homepage:

- **app-store-badge.svg** — Apple “Download on the App Store” (Black lockup), from `Download-on-the-App-Store/US/Download_on_App_Store/Black_lockup/SVG/`.
- **google-play-badge.png** — Google “Get it on Google Play” (Digital/Web color English), from `Google Play Badge guidelines/Get it on Google Play Badges/Digital/png/`.

Both badges are displayed in the same size (130×40px) so they align. To replace or update, copy the correct file from the official guideline folders and overwrite the file here with the same filename.
