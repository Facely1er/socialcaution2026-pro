## Fields Added

All services now include the following optional fields where applicable:

- `data_export_format` - Formats available for data export
- `data_broker_sales` - Whether data is sold to data brokers
- `law_enforcement_sharing` - Whether data is shared with law enforcement
- `affiliate_sharing` - Whether data is shared with affiliates
- `government_requests` - Number of government data requests
- `right_to_portability` - GDPR right to data portability
- `right_to_rectification` - GDPR right to rectification
- `opt_out_mechanisms` - How users can opt out of data sharing
- `password_requirements` - Password strength requirements
- `security_incidents` - Number of security incidents
- `vulnerability_disclosure` - Vulnerability disclosure policy
- `security_headers` - Security headers implementation
