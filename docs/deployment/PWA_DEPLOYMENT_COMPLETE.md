## 📱 How Users Install the PWA

### On Android (Chrome)
1. Visit https://socialcaution.netlify.app
2. Browser shows "Add to Home Screen" banner
3. Tap "Add" or use menu → "Install App"
4. App installs and appears on home screen
5. Opens in standalone mode (no browser UI)

### On iOS (Safari)
1. Visit https://socialcaution.netlify.app
2. Tap Share button (square with arrow)
3. Select "Add to Home Screen"
4. Customize name (optional)
5. Tap "Add"
6. App appears on home screen
7. Opens in standalone mode

### On Desktop (Chrome/Edge)
1. Visit https://socialcaution.netlify.app
2. Look for install icon in address bar
3. Click "Install" or use menu → "Install SocialCaution"
4. App installs and appears in apps/taskbar
5. Opens in standalone window
